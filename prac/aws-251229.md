# AWS Bedrock 기반 LLM에서 AI Agent까지
## Amazon Bedrock Knowledge Bases
- RAG(Retrieval Augmented Generation) 기술을 활용하여 사용자의 개인 데이터를 GenAI 애플리케이션에 통합할 수 있는 서비스
- 데이터 소스에서 관련 정보를 검색하여 생성된 응답의 관련성과 정확성 향상
