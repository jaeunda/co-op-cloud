# Soongsil Univ. Co-op Cloud Track (2025 Winter)
**2025학년도 겨울학기 Co-op AI 클라우드 트랙**의 학습 및 실습 기록입니다.
## 개요
- 기간: 2025.12.22. ~ 2026.1.14.
- 주관: 스파르탄SW교육원
- 핵심 목표
	- 클라우드 네이티브(Cloud Native) 개념과 PaaS 플랫폼 아키텍처 이해
	- K-PaaS(Open Cloud Platform) 설치 및 운영 실습 
	- AWS 인프라 설계 및 Kubernetes 오케스트레이션 구현 
	- MSA(Microservices Architecture) 및 CI/CD 파이프라인 구축
```text
co-op-cloud/
├── notes/      # 이론 정리
│   └── ...
├── prac/       # 실습 정리, troubleshooting 로그
│   └── ...
└── README.md
```
- 리소스 배포를 넘어 기술의 기반 원리를 이해하는 데 중점을 두었습니다.
